Hàm IF lồng nhau: Xử lý nhiều điều kiện phức tạp dễ dàng

Đăng bởi: Admin | Chuyên mục: Hàm Logic

Hàm IF cơ bản chỉ giải quyết được bài toán "Đúng hoặc Sai". Nhưng nếu bạn cần xếp loại học sinh thành 4 mức (Giỏi, Khá, Trung bình, Yếu) thì sao? Đó là lúc bạn cần dùng Hàm IF lồng nhau (Nested IF).

[Quảng cáo Google AdSense - Top Banner]

1. Cấu trúc hàm IF lồng nhau

Hiểu đơn giản: Bạn đặt một hàm IF khác vào bên trong phần "Giá trị nếu sai" của hàm IF đầu tiên.

=IF(Điều_kiện_1, Kết_quả_1, IF(Điều_kiện_2, Kết_quả_2, Kết_quả_còn_lại))

Nguyên tắc hoạt động: Excel sẽ kiểm tra lần lượt từ trái sang phải. Nếu Điều kiện 1 đúng, nó lấy Kết quả 1 và dừng lại. Nếu sai, nó mới xét tiếp đến Điều kiện 2.

2. Ví dụ thực tế: Xếp loại học lực

Giả sử bạn có điểm số ở ô B2 và cần xếp loại theo quy tắc:

  • Điểm >= 8: Giỏi
  • Điểm >= 6.5: Khá
  • Điểm >= 5: Trung bình
  • Còn lại: Yếu
[Ảnh minh họa bảng điểm học sinh]

Công thức sẽ là:

=IF(B2>=8, "Giỏi", IF(B2>=6.5, "Khá", IF(B2>=5, "Trung bình", "Yếu")))

3. Sai lầm chết người khi dùng IF lồng nhau

Rất nhiều bạn viết sai thứ tự logic. Ví dụ: Bạn xét điều kiện >5 trước điều kiện >8.

Ví dụ sai: =IF(B2>=5, "Trung bình", IF(B2>=8, "Giỏi"...))

=> Kết quả sai: Nếu điểm là 9, Excel thấy 9 > 5 là đúng nên trả về "Trung bình" ngay lập tức mà không xét tiếp.

Mẹo nhớ: Khi dùng dấu Lớn hơn (>=), hãy xét từ Lớn nhất xuống Nhỏ nhất.

4. Giải pháp hiện đại: Hàm IFS (Nên dùng)

Nếu bạn dùng Excel 2019 hoặc Office 365, hãy quên IF lồng nhau đi. Hàm IFS viết ngắn gọn hơn nhiều:

=IFS(B2>=8, "Giỏi", B2>=6.5, "Khá", B2>=5, "Trung bình", TRUE, "Yếu")
